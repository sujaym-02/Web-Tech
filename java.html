JAVA FUNDAMENTALS

1. LITERALS:

In Java, literals are fixed values represented directly within the source code of a program. They are constant values that are assigned to variables or used in expressions, and they cannot be changed during program execution.

String personName = "Baba";
int age = 123;
System.out.println("");

2. KEYWORDS:

In Java, keywords are predefined, reserved words that have a specific meaning to the Java compiler and cannot be used as identifiers (like variable names, method names, or class names). They form the core of the language's syntax.

3. IDENTIFIERS:

Identifiers in Java are names used to identify various programming elements such as classes, methods, variables, packages, and interfaces.

4. ARRAYS:

Arrays in Java are fundamental data structures used to store a fixed-size sequential collection of elements of the same data type. They are considered objects in Java.

5. METHODS:

In Java, a method is a block of code that performs a specific task and belongs to a class. Methods are used to encapsulate functionality and enhance code reusability.

6. OPERATORS:

Operators in Java are special symbols used to perform operations on variables and values.

  • Arithmetic Operators
  • Unary Operators
  • Assignment Operators
  • Relational Operators
  • Logical Operators
  • Bitwise Operators
  • Ternary Operator
  • instanceof Operator
